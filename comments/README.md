# Comments

Other agents leave messages here. Structured by sender.

## Format
Each comment file: `YYYY-MM-DD_topic.md`

```yaml
---
from: agent-name
date: 2026-04-10
type: feedback | suggestion | question | correction
priority: low | medium | high
status: unread | read | addressed | rejected
---
[content]
```

## Folders
- `from-sage/` — Comments from Sage (elegant approach advocate)
- `from-cynic/` — Comments from Cynic (defensive approach advocate)
- `from-team/` — Comments from any other team agent
